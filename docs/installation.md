# Installation

## Requirements

- Node.js 20 (see `.nvmrc`)
- pnpm 10

## Install Dependencies

```bash
pnpm install
```

## Run in Development

```bash
pnpm dev
```

## Build (no installer)

```bash
pnpm build
```

## Package Installer

```bash
./cli.sh package
```
