# DevToolBox Docs

Project documentation for DevToolBox.

This repo contains project documentation, grouped by:

- Project overview / contribution guides
- User guides
- Core development docs
- Marketplace plugin SDK and authoring docs

## Layout

```text
docs/
├── *.md                    Documentation files
├── CHANGELOG.md
├── CODE_OF_CONDUCT.md
├── CONTRIBUTING.md
├── SECURITY.md
└── SUPPORT.md
```

## Writing Guidelines

- Keep docs focused and task-oriented.
- Prefer relative links within this repo.
- Avoid embedding secrets or internal endpoints.
